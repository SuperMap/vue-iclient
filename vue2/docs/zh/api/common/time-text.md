# 时间

```vue
<sm-time-text :fontStyle="{ fontSize: '18', fontWeight: '700' }" timeType="date+second+week">
</sm-time-text>
```

### Attributes

| 参数      | 说明     | 类型   | 可选值                                        | 默认值 |
| :-------- | :------- | :----- | :-------------------------------------------- | :----- |
| timeType  | 时间类型 | string | 'date' \| 'date+second' \| 'date+second+week' | 'date' |
| fontStyle | 字体样式 | object | -                                             | -      |

> 支持[主题混入参数](/zh/api/mixin/mixin.md#theme)
